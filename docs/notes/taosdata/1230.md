## 进行 python 版本管理

- brew install pyenv
- 添加配置

```bash
export PATH="$HOME/.pyenv/bin:$PATH"
eval "$(pyenv init --path)"
eval "$(pyenv virtualenv-init -)"
```

- 安装： pyenv install 3.12.3
- 设置全局版本： pyenv global 3.12.3
