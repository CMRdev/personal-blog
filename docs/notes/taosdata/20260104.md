# 日记

## 一、`status history`面板

- 1、右侧图形配置
- 2、展示逻辑：采样点缺失、采样间隔不同
- 3、数据是个有限`离散值`

### 渲染图表所需参数

| 属性      |                    类型 | 说明                       | 示例            |
| --------- | ----------------------: | -------------------------- | --------------- |
| name      |                  string | 监控的实体名称或标识符             | `电压`          |
| timestamp |                  number | 时间戳，采样时间(包含空值) | `1672790400000` |
| status    | string、number、boolean | 状态值                     | `true｜false`   |

```bash
# 当某数据不存在时（ps：不存在不代表值为 0），echarts可以用 '-' 或者 null 或者 undefined 或者 NaN 表示。
# 采样间隔不同时，方案一：按时间正常渲染；方案二：获取所有采样点的时间，在x轴上均匀显示，忽略时间概念
```
