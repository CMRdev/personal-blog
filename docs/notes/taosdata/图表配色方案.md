## 图表配色方案

- Trends、bar、scatter等多系列图表
  - Color Schema：默认选中 `Classic palette`，每个系列分配不同颜色
  - Value Mappings： 配置的文字、颜色只影响`ToolTip`,不影响线条的配色
  - Thresholds：当ColorSchema选择从Thresholds取色时，thresholds的配置`应用于每个系列的Last值`，匹配上则改变整个系列的颜色

- stats
  - 单系列
    - value：取最后的`Last值`
  - 多系列
    - value: 取单系列最后的`Last值`

  - Color Schema：默认选中 `From Thresholds`
  - Value Mappings： 配置的文字、颜色影响`stats的背景色`及`数值显示`
  - Thresholds：当ColorSchema选择从Thresholds取色时，thresholds的配置`应用于每个系列的Last值`，匹配上则改变整个系列的颜色

- Gauge图表
  - 单系列
    - start：取所有数据的`最小值`
    - end：取所有数据的`最大值`
    - value：取最后的`Last值`
  - 多系列
    - start：取所有系列的数据的`最小值`
    - end：取所有系列的数据的`最大值`
    - value：取单系列最后的`Last值`

  - Color Schema：默认选中`From Thresholds`
  - Thresholds：所有系列使用同一个Thresholds配色方案，`无法对单系列单独配色`
  - Value Mappings: 配置的文字、颜色只影响`Gauge数值的文字、颜色显示`,不影响Gauge边框的颜色

- BarGauge
  - 单系列
    - start：取所有数据的`最小值`
    - end：取所有数据的`最大值`

  - 多系列
    - `一个柱条代表一个系列`
    - min：取所有系列的数据的`最小值`
    - max：取所有系列的数据的`最大值`
    - value：取单系列最后的`Last值`
  - Color Schema：默认选中`From Thresholds`
  - Thresholds：所有系列使用同一个Thresholds配色方案，`无法对单系列单独配色`
  - Value Mappings: 配置的文字、颜色只影响`Gauge数值的文字、颜色显示`,不影响barGauge柱条的颜色
