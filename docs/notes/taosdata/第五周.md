## 查询

- 现在查询 tag 信息和值的方式总共有三种：
  - show tags from 子表名
  - select \* from information_schema.ins_tags where
  - SELECT TAGS
- 在数据库中子表数量较大的时候（百万级）前两种方式的性能会变得非常差

## panel 和 panelTemplate 排序规则设计

- `元素浏览器-面板`【普通面板和面板模版有各自的排序规则，不可交叉】
  - 普通面板：有`面板的编辑`权限才可以`上移下移`和`拖拽排序`
  - 面板模版：有`模版的编辑`权限才可以`上移下移`和`拖拽排序`
- `元素浏览器-仪表板`
  - 普通仪表板：有`模版的编辑`权限才可以设置`置顶`
  - 仪表板模板：无置顶功能
- `基础库-面板模版`
  - 暂不改动，考虑到使用场景不支持也无需`上移下移排序`
- `基础库-仪表板模版`
  - 仪表板模版：有`模版的编辑`权限才可以设置`置顶`
