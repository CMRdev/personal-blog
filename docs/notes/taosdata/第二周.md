## 日记

- TS ？ ‌Technical Specification / Test Support
  试验规格，测试报告
- PR ？Pull Request
  - 用于向项目维护者提议合并代码变更。开发者通过创建 Pull Request 来审查、讨论和最终整合代码修改
- FS ? ‌Functional Specification‌
  - 功能规格说明书
- JIRA
  一个项目管理与缺陷跟踪工具
  核心概念：

  - Issue）：涵盖任务、缺陷、需求等，可自定义字段和状态
  - 工作流 ‌：支持自定义问题处理流程（如“新建 → 处理中 → 解决 → 关闭”）
  - 看板与冲刺 ‌：提供可视化工具管理迭代周期（Sprint）和任务看板
