# 改进

- 首次加载请求的资源太多了（未验证打包发版时是否对资源进行了打包），导致页面卡顿严重，可以思考如何优化

- 面板的AI提示生成的内容占据了太大空间了，可以考虑分屏横向布局，用户体验可能会有提高

- panels的各种echarts图表，代码耦合比较严重，可适当拆分，方便维护

- 项目中的form表单里的每一项都是双行显示，`文字提示单占一行`，视觉上看起来很乱，可考虑单行显示
    - 考虑到中英切换，可开发控件，鼠标悬浮时自动展开文字提示
    - 文字提示（label）可加上背景色

- 有些页面的空白太多

- 注册流程分成了两个页面

- 元素树层次太多，需要点击好几层才找到目标，可添加层级展开功能和搜索功能
