## 2. 键盘快捷键

| 快捷键                       | 说明                                         |
| ---------------------------- | -------------------------------------------- |
| Ctrl+V                       | 直接粘贴截图（Mac 用户注意是 Ctrl 不是 Cmd） |
| Ctrl+J / Option+Enter（Mac） | 换行                                         |
| Ctrl+R                       | 搜索历史 prompt                              |
| Ctrl+U                       | 清空整行输入                                 |
| Esc Esc（双击）              | 退出代码编辑，回到对话                       |

## 3. Slash 命令

| 命令                   | 说明                                                                      |
| ---------------------- | ------------------------------------------------------------------------- |
| /simplify              | 启动 3 个并行 Agent，分别从代码复用、代码质量、运行效率三个角度审查改动   |
| /btw                   | 并行分支对话                                                              |
| /branch                | 将当前对话分叉为两个并行会话，各走一边，最后挑效果好的（平行宇宙）        |
| /rewind                | 回退代码，保留对话（后悔药）                                              |
| /insights              | 观察自己使用 Claude 的习惯                                                |
| /model opusplan        | 切换模型（可在 Opus 4.6、Sonnet、Haiku 等间切换，还能调整模型"用力程度"） |
| /context               | 查看当前上下文窗口的使用情况，包括令牌消耗等                              |
| /clear                 | 重置上下文窗口                                                            |
| /loop 5m <任务>        | 每隔指定时间循环执行任务，默认间隔 10 分钟                                |
| /rc 或 /remote-control | 生成一个远程控制 URL                                                      |
| /export                | 将当前整段对话导出为 Markdown 文件                                        |

**新功能：** agent-team
