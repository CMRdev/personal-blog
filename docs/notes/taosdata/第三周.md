## 面板绘图：自动设定元素属性的极限值

- 读取属性的限值配置【Minimum、LoLo、Lo、Target、Hi、HiHi、Maximum】
- 面板配置-坐标轴自动带入数值范围，匹配规则：
  - 只有一个指标
    - 读取限值配置，取所有配置项的值，获取`min`、`max`
  - 多个指标
    - 每个指标都做了限值配置最大最小值
      - 取所有数据的最大值为`max`、取所有数据的最小值为`min`
    - 有任何一个指标没配置最大值或最小值
      - 对应的，不配置`max`或`min`

- 右坐标轴开启的情况
  - 查验右坐标轴系列指标【总指标 - 右坐标轴系列指标 = 左坐标轴系列指标】
  - 同样适用于上面的匹配规则
