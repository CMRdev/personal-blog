
- /boot/ 系统启动
- /bin/ 系统命令
- /sbin/ 系统环境设置命令，只有root用户可用
- /dev/ 设备文件
- /etc/ 配置文件（系统内所有采用rpm安装的服务配置文件，如用户信息、服务的启动脚本、常用服务的配置文件）
- /home/ 普通用户的主目录（家目录）
- /root/ root用户的主目录
- /lib/ 系统调用的函数库
- /media/ 挂载媒体设备，软盘、光盘
- /mnt/ 挂载目录，U盘、移动硬盘、其他操作系统的分区
- /misc/ 挂载目录，挂载NFS服务的共享目录
- /opt/ 第三方软件安装目录【也可安装到/usr/local下】
- /srv/ 服务数据目录
- /tmp/ 临时目录，不要保存重要数据，最好每次开机都把该目录清空
