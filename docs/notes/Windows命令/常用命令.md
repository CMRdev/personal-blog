# windows 常用命令

## 1、打开远程桌面

```bash
win+R
mstsc (Microsoft terminal services client)
```

## 2、打开任务管理器

```bash
win+R
taskmgr
```

## 3、查看电脑配置

```bash
win+R
dxdiag
```

## 4、打开注册表

```bash
win+R
regedit # 助记：reg-edit
```

## 5、打开剪切板历史记录

```bash
win+v # visible
```

## 6、最小化本屏幕的所有窗口

```bash
win+m # 没有win+d威力大
```

## 7、打开 windows 设置

```bash
win+i
```

## 8、打开资源管理器

```bash
win+e
```
